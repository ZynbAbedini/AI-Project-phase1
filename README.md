Implementation of **Single-Variable Linear Regression from scratch** as part of an Artificial Intelligence course assignment.

## 🎯 Objective

Predict Pokémon `Attack` using the `Defense` feature and analyze optimization behavior using:

- Mean Squared Error (MSE)
- Grid Search
- Gradient Descent
- Feature Normalization

## 🔍 Key Insights

- Large learning rates → divergence  
- Small learning rates → slow convergence  
- Normalization → faster and more stable training  
- Gradient Descent outperformed limited Grid Search  

## 🛠 Tech Stack

Python • NumPy • Pandas • Matplotlib • Jupyter Notebook

## 📌 Concepts Practiced

Optimization, Learning Rate Tuning, Convergence Analysis, Data Scaling
